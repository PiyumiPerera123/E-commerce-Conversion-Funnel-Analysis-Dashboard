E-commerce Conversion Funnel Analysis

This project analyzes user behavior in an e-commerce funnel using Python and Power BI to identify drop-offs and improve conversion rates.

Funnel Flow: Browse → Add to Cart → Checkout → Purchase

Key Metrics
Conversion Rate = Purchases / Total Sessions
Drop-Off Rate = Loss between funnel stages
Bounce Rate = Bounce Sessions / Total Sessions

Tools Used
Python, Pandas, NumPy, Plotly, Matplotlib, Power BI, DAX

Key Insights
Major drop-off at Checkout → Purchase stage
Email and Google Ads performed best
Mobile and Desktop usage were similar
East region generated highest revenue
